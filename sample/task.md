# statement

非負整数 $A,B$ が与えられるので $A+B$ を出力してください。

# input

入力は以下の形式で標準入力から与えられます。

```
$A$ $B$
```

- 入力は全て整数で与えられる
- $0\leq A,B\leq 10 ^ {12}$

# sample

```sample-input
1 1
```

```sample-output
2
```

```sample-input
2 2
```

```sample-output
4
```

```sample-input
3 3
```

```sample-output
6
```
